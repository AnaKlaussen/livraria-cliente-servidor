# Livraria - Arquitetura Cliente-Servidor

## Descrição
Este projeto refatora o aplicativo da livraria para uma arquitetura Cliente-Servidor.

## Estrutura do Projeto

- `frontend/` - Interface do usuário (Cliente)
- `api_gateway/` - API Gateway com Flask (Servidor)
- `services/` - Lógica de negócio da aplicação
- `database/` - Estrutura preparada para armazenamento de dados

## Tecnologias Utilizadas

- Python
- Flask
- HTML
